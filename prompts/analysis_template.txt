# 1. 一句话定位

# 2. 导师式速读
- 先用通俗语言讲：作者发现了什么痛点？
- 再讲研究问题：输入是什么、输出是什么、优化目标或评价目标是什么？

# 3. 背景铺垫
- 读懂本文前需要哪些前置知识？

# 4. 方法拆解
- 方法总流程：按步骤拆成可执行的pipeline。

# 5. 创新点与真实贡献

# 6. 实验与证据链
